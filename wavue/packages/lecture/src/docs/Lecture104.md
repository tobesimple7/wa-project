# NVM(Node Version Manager) 

## 1. NVM이란?
---
### - **NVM**은 Node.js의 **버전 관리 도구**입니다.  
### - 하나의 PC에서 **여러 버전의 Node.js**를 설치하고 자유롭게 전환할 수 있게 해줍니다.  
### - 예를 들어, Vue 2는 Node 16 버전이 필요하고 Vue 3은 Node 20 버전을 사용할 수도 있습니다.  
### - 이럴 때 NVM을 이용하면 **버전 충돌 없이 프로젝트별로 다른 Node 버전**을 사용할 수 있습니다.

## 2. NVM의 역할
---
### **Node.js 설치 및 관리** : 다양한 Node.js 버전을 쉽게 설치/삭제 
### **버전 전환** : 명령 한 줄로 Node.js 버전 전환 (`nvm use 20`) 
### **프로젝트별 버전 고정** : `.nvmrc` 파일을 이용해 프로젝트별 버전 자동 설정 
### **충돌 방지** : 여러 프로젝트에서 다른 Node 버전을 동시에 관리 가능
