## 1.환경설정

### node.js : 자바스크립트를 c#, java 처럼 사용할 수 있도록 한 개발툴 

### npm : 라이브러리 관리, 배포관리(http://npmjs.com)
        
### pnpm : npm 과 같지만, 효율적인 라이브러리 관리

### 프로젝트 설정 : vue 프로젝트를 package.json이 관리

### vite 설정 : vue로 만든 프로그램을 vite가 컴파일(빌드()함. vite.config.json(vite 설정파일))

### 타입스크립트 설정: 타입스크립트를 사용할 경우, tsconfig.json 파일을 설정해야 함

### WaVue 프로젝트(모노레포) : 강의 소스가 있는 프로젝트(모노레포 구조로 되어 있음)

### 모노레포: 하나의 프로젝트에 2개 이상의 프로젝트가 있는 구조
