# PNPM (Performant NPM) 이란?

## 1. PNPM이란?

### **pnpm**은 Node.js의 또 다른 패키지 관리 도구입니다.  
### (NPM, Yarn과 같은 역할을 하는 **고성능 패키지 매니저**)
### NPM보다 **속도가 빠르고**, **디스크 공간 절약**에 강점이 있습니다.
### Node.js를 설치한 후, 설치 명령을 통해 사용 가능합니다.
### Vue.js, React.js, Express.js 등 대부분의 JavaScript 프로젝트에서 사용할 수 있습니다.
### 모노레포(하나의 프로젝트에 2개 이상의 프로젝트를 가진 구조) 환경에서 사용하기 좋습니다.

### pnpm 설치 : npm install -g pnpm   
### pnpm 버전 확인 : pnpm -v          
### 패키지 설치 : pnpm add 패키지명
