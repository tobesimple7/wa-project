# NPM(Node Package Manager) 이란?

## 1. NPM이란?
---
### - **npm**은 **Node.js의 패키지 관리 도구**(Node Package Manager)입니다.  
### - 프로그램 개발에 필요한 **라이브러리(패키지)** 를 쉽게 설치하고 관리할 수 있습니다.  
### - **Node.js를 설치하면 npm이 자동으로 함께 설치됩니다.**  
### - Vue.js, React.js, Express.js 등 거의 모든 자바스크립트 기반 프로젝트에서 사용됩니다.  
### - npmjs.com 살펴보기 : http://npmjs.com

## 2. NPM의 역할
---
### **라이브러리 설치** : 다른 개발자가 만들어 둔 기능(모듈)을 쉽게 설치 
### **버전 관리** : 각 패키지 버전별로 의존성(Dependencies)을 관리 
### **스크립트 실행** : 개발 서버 실행, 빌드 등 명령을 자동화 (`npm run dev`) 
### **패키지 관리 파일 생성** : `package.json` 파일을 통해 프로젝트 정보를 관리 

### npm 버전 확인 : npm -v    
### 라이브러리 설치 : npm install 라이브러리명
