# VSCode 편집기

## 1. VSCode란?
---
### - **VSCode(Visual Studio Code)**는 마이크로소프트에서 만든 **무료 오픈소스 개발 도구**입니다.  
### - **Node.js, Vue.js, React, Python, Java** 등 다양한 언어 개발에 사용할 수 있습니다.  
### - **가볍지만 강력한 확장 기능(Extensions)** 으로 IDE 수준의 개발 환경을 제공합니다.  
### - **실시간 디버깅, Git 통합, 코드 자동완성(IntelliSense)** 등을 지원합니다.  

## 2. VSCode 다운로드 및 설치
---
### VSCode 공식 사이트  https://code.visualstudio.com
### **운영체제(Windows / macOS / Linux)** 에 맞는 버전을 선택하여 다운로드합니다.
